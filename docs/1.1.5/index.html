Descrição

O componente gumgaList foi criado no intuito de substituir a gumgaTable. A partir de uma fonte de dados, é gerado uma tabela que é configurada através de um objeto javascript.

Instalação

// npm
npm i gumga-list-ng --save

// bower
bower install gumga-list-ng --save

Demonstração

Uso

  • HTML
  • JavaScript
<gumga-list
  data="ctrl.exemplo"
  configuration="ctrl.tableConfig">
</gumga-list>
ctrl.tableConfig = {
  columns: 'tag_name, published_at',
  checkbox: true,
  materialTheme:true,
  columnsConfig: []
}
